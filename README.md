# HealthAI Coach — Pack complet (Front + Backend) — Port 8010

## Démarrage (Windows)
1) Dans `backend/healthai-backend/`, lance **setup_windows.bat** (une seule fois)
2) Double-clique **start_all.bat**

## URLs
- Backend: http://localhost:8010
- Swagger: http://localhost:8010/docs
- Front:   http://localhost:5173

## Config API (Front)
Le front est déjà configuré avec:
`VITE_API_BASE_URL=http://localhost:8010`
dans `front/healthai-front/.env`.
