Pipeline ETL — Automatisation & Qualité

📥

Ingestion automatisée

  • APScheduler — tâches Cron
  • sync_food_data → 00h30 quotidien
  • sync_fitness_data → 01h00 quotidien
  • data_quality_check → toutes les heures
🧹

Nettoyage & Validation

  • Suppression des doublons (email, ID)
  • Imputation des valeurs nulles (médiane)
  • Validation des plages (CHECK SQL)
  • Normalisation des formats (ISO dates)
📊

Gestion de la qualité

  • Table EtlRun — chaque exécution tracée
  • Table EtlError — anomalies enregistrées
  • Métriques : rows_in, rows_out, errors
  • Logs JSON structurés
✅

Règles qualité appliquées

Complétude≥ 95 %
Unicité0 doublon
Validité plages≤ 1 % hors borne
